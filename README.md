Hey all! I'm back with a mod for the PC port of Megaman: Phantom of The Network.
This mod will NOT work on the original/ emulated version.

This mod replaces all sound effects and music with more fitting sounds or remixes.
I'm proud to say this mod completely changed my experence with the game. It's crazy what a little music and sound can do.

To install the mod, simply open the assets folder and extract the folders from the ZIP file there. Let it overwrite the files inside and you are good to go.
